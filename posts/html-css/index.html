前篇

Blog这东西前面的文章也说了，很烦人其实，尤其是配置文件的时候，老是出错；最近看Hexo主题的时候又看上

了一个主题——Matery；这款主题页面清晰，画风很好，功能很多，想着就换了，但是因为不会前端的内容，里

前端三件套 = HTML + CSS + JS；这些是web开发需要学的东西，非常好玩；

这篇博客就整理我的学习历程，先开始的HTML，所以先整理HTML；

IDE

有人会问，编写前端这些代码需要什么IDE，其实这个不需要太大的IDE就行；最简单的是记事本，哈哈哈，没想

到吧；想着也是，把后缀改成html就行了，还可以用Notepad++；我也在网上找了几款，比如说像Vscode，

jetbrains公司的webstorm等等，这些都可以用，我也问了几个朋友，挑来挑去，咦~，发现了一款非常好的软件

——HBuilderX；

先说说它的好处，这款软件内存特别小，下载就几十MB，非常方便，它可以直接生成一个本地端口（应该算

是），就可以浏览你写的网页的样式；最重要的一个优点是，他可以代码补全，对于初学者来说，真的不错；

div+span标签

● div（块）：< div >标签用来划分一个区域，相当于一块区域容器，可以容纳段落、标题、表格、图像等各种网

页元素；HTML中大多数的标签都可以嵌套在< div >标签中，< div >中还可以嵌套多层< div >，用来将网页分割成

独立的、不同的部分，来实现网页的规划和布局。

● span（内联）：< span >标签用来修饰文字的，div与span都是没有任何默认样式的，需要配合CSS来使用。

CSS颜色表示

1、单词表示法

用英文单词直接表示（看你单词水平厉不厉害）

2、16进制表示法

表示方法：‘#’ + 6位16进制数

取值范围：#000000 – #ffffff

3、RGB三原色表示法(红绿蓝)

表示方法：rgb(x,y,z)

x,y,z取值范围：0~255

CSS背景样式

background-color：背景颜色

background-image：背景图

background-repeat：背景图片的平铺方

background-position：背景图片的位置

background-attachment：背景图随滚动条的移动方式

1、background-color：背景颜色

2、background-image：背景图片

​ ● url（背景地址）

​ ● 默认：水平垂直铺满背景图

3、background-repeat：背景图片的平铺方式

​ ● repeat-x：x轴平铺

​ ● repeat-y：y轴平铺

​ ● repeat：x，y都平铺（默认值）

​ ● no-repeat：x，y都不平铺

4、background-position：背景图片的位置

格式：x y

​ ● x和y可以是数字（单位px），可以用百分数

​ ● x和y可以是英文单词，left、center、right、top、bottom…随意搭配

5、background-attachment：背景图随滚动条的移动方式 常用值：

​ ● scroll：默认值：背景元素按照当前元素进行偏移

​ ● fixed：背景位置是按照浏览器进行偏移

border-style：边框的样式

border-width：边框的大小

border-color：边框的颜色

1、border-style：

● solid：实线

● dashed：虚线

● dotted：点线

CSS文字样式

1、font-family: 字体样式

● 英文字体：Arial、’Times New Roman‘…

● 中文字体：楷体、宋体…

常用的中文字体对应的英文名称：

中文 英文
宋体 SimSun
黑体 SimHei
微软雅黑 Microsoft Yahei
楷体 KaiTi
新宋体 NSimSun
仿宋 FangSong

2、font-size: 字体大小

大小的单位有px、rem

也可以取值：

属性取值 字体大小
xx-small 最小
x-small 较小
small 小
medium 正常（默认大小，为16px）
large 大
x-large 较大
xx-large 最大

3、font-weight：字体粗细

粗细：normal（正常）、bold（加粗）

或者数字：100、200…900（100—500正常，600—900加粗）

6、font-style：字体样式

样式：normal（正常）、italic & oblique（斜体）

CSS段落样式

1、text-decoration：文本修饰

● 下划线：underline

● 上划线：overline

● 删除线：line-through

● 无修饰：none

可以添加多个文本修饰，中间用空格隔开

2、text-transform：文本大小写（英文段落）

● 大写：uppercase

● 小写：lowercase

● 只首字母大写：capitalize

3、text-indent：文本缩进

text-indent：数值，单位px或者em（em是相对单位，1em默认和字体大小相同）

4、text-align：文本对齐方式

● 左对齐：left

● 右对齐：right

● 居中：center

● 两端点对齐：justify

5、line-height：定义行高

可以是px单位数字，也可以纯数字（跟字大小成比例）

6、letter-spacing：定义字间距

7、word-spacing：定义词间距（英文）
